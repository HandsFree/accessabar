# Methods
List of ACE's methods.

## open()
Opens ACE in the configured position.

Example:
```javascript
import Accessabar from '@handsfree/accessabar';

const accessabar = new Accessabar();

accessabar.open();
```

## close()
Closes ACE.

Example:
```javascript
import Accessabar from '@handsfree/accessabar';

const accessabar = new Accessabar();

accessabar.close();
```
