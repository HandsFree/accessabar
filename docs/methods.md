# Methods
List of Accessabar's methods.

## open()
Opens Accessabar in the configured position.

Example:
```javascript
import Accessabar from '@handsfree/accessabar';

const accessabar = new Accessabar();

accessabar.open();
```

## close()
Closes Accessabar.

Example:
```javascript
import Accessabar from '@handsfree/accessabar';

const accessabar = new Accessabar();

accessabar.close();
```
